# TCC-Projeto-EAD_frontend
Trabalho de conclusão do curso sistemas de informação

## Requirements
- Node.js
- Yarn ou Npm

## Getting Started
1° - Clonar repositorio
$ git clone "https://github.com/zagob/TCC-Projeto-EAD_backend.git"

2° - Instalar dependências
$ yarn

3° - Executar projeto
$ yarn start
